## Descripción 

Descripción del Trabajo Práctico: Algoritmia y Entornos de Desarrollo
PSeInt (Diseño y Lógica): Se utilizaron herramientas de pseudocódigo y diagramas de flujo para el planteo lógico de los problemas. El enfoque estuvo en dominar las estructuras de control básicas (condicionales y bucles) de una manera visual y conceptual, asegurando que la lógica algorítmica fuera correcta antes de pasar al código real.

Visual Studio Code (Codificación y Práctica Profesional): Se migró el diseño de los algoritmos a un entorno de desarrollo profesional (IDE). Aquí nos enfocamos en la sintaxis limpia, la estructura del código fuente, el uso de comentarios técnicos y la ejecución/depuración real de los programas.
